# StarWarsApi
## Desafio Técnico para um processo seletivo


## Description
> Desenvolver uma API REST que retorne informações sobre os planetas do universo Star Wars.
Utilizar Python, Flask e MongoDB.

## Usage
```bash
$ docker-compose up
```

## Endpoints
- GET /planets
- GET /planets/id
- POST /planets
- DELETE /planets/id
- PUT /planets/id
---
- GET /movies
- GET /movies/id
- POST /movies
- DELETE /movies/id
- PUT /movies/id

## Swagger
- http://localhost:8081/

## Run linters and tests
```bash
$ cd app/
$ make
```
